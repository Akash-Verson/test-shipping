This sample showcases threaded comments using shipment data. Add remarks and replies to cells, preview comments on hover, and manage threads from an optional review pane.

The Comments feature lets you add remarks and threaded replies to cells. When a cell contains a comment, a small indicator appears in its corner. Hover over the indicator to view the comment. You can show or hide the comments review pane using the showCommentsPane property:

  • false (default) - Hides the review pane.
  • true - Displays the review pane.

Comments can be added directly to cells via the context menu, the ribbon, the comments review pane, or programmatically using the updateCell method. You can perform the following actions:

  • Add Comment - Create a new comment to provide feedback or context.
  • Edit Comment - Update the text of a comment or reply to keep information accurate.
  • Delete Comment - Remove a single comment, a reply, or the entire thread when it's no longer needed.
  • Reply - Add one or more replies to an existing comment thread to provide additional details or answers.
  • Resolve - Mark a comment thread as resolved when the issue has been addressed.
  • Reopen - Restore a resolved thread if further discussion or clarification is required.
