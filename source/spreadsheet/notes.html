This sample demonstrates the Notes feature using the shopping cart scenario as an example.

The Notes feature lets you add annotations to cells while reviewing a worksheet. When a cell contains a note, a red indicator appears in the corner. Hover over the indicator to view the note. You can enable or disable notes using the enableNotes property; by default, it's enabled.

Notes can be added directly to cells using the context menu, the ribbon, or programmatically using the updateCell method. You can perform the following actions:

  • Add Note - Create a new note for a cell to provide feedback or context.
  • Edit Note - Update the text of an existing note to keep information accurate.
  • Delete Note - Remove a note when it's no longer needed.
  • Show/Hide Note - Toggle the visibility of the note on the selected cell.
  • Previous Note - Navigate to the previous note in the workbook.
  • Next Note - Navigate to the next note in the workbook.
  • Show All Notes - Display all notes in the worksheet for a comprehensive view.

The following keyboard shortcuts apply to the Notes feature:

  • Shift + F2 - Add a note to the current cell.
  • Esc - Save and close the note only if the isVisible property is set to false.
